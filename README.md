<h1 align="center"> University Database </h1>

Database that contains a records of Students and Employess:

### Student Records

 - First name 
 - Last name
 - Address
 - Index
 - number PESEL 
 - Gender

### Employee Records

 - First name 
 - Last name 
 - PESEL number 
 - Gender 
 - Address 
 - Salary

### Functions

-  Display the entire database
-  Add Student
-  Add Employee
-  Search by PESEL 
-  Search by Name
-  Sort by PESEL
-  Sort by Name
-  Deleate by Index
-  Adding a generate Student
-  Adding a generate Employee
-  Modify Salary
-  Sort by salary
-  Addd Database from file
-  Save All Database to file
<h1 align="center">How to use</h1>


### Building project
    mkdir build
    cd build
    cmake ..
    make -j
